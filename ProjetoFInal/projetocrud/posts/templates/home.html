Bem-vindo ao Projeto CRUD de Posts!

Este é um projeto Django simples que demonstra um CRUD (Criar, Ler, Atualizar e Deletar) de posts, utilizando uma API externa (JSONPlaceholder) para sincronizar os dados com um banco de dados local.

Use os links abaixo para navegar pelas funcionalidades do projeto:

  • Visualizar e Gerenciar os Posts
    Veja a lista de todos os posts disponíveis em nosso banco de dados. Você poderá editar ou excluir qualquer um deles.

  • Adicionar Novo Post
    Crie um novo post que será salvo no nosso banco de dados.

  • Sincronizar Dados da API
    Use este link para popular o seu banco de dados com os dados da API JSONPlaceholder. Esta ação pode ser feita uma única vez para iniciar o projeto.
